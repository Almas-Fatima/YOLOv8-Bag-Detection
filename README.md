
# 👜 YOLOv8 Bag Detection System

## 📌 Overview
This project implements a real-time object detection system using YOLOv8 to classify different types of bags into two categories:
- Fashion Bags (purses, handbags)
- Shopping Bags (carry bags)

## 🎯 Features
- Real-time object detection
- Custom-built dataset
- Accurate classification using YOLOv8

## 🧠 Model
- YOLOv8 (Ultralytics)
- Trained on a custom dataset

## 📊 Dataset
Due to size limitations, the dataset is not included in this repository.

Data was collected from:
- Real-world images
- Internet sources
- Annotated using Roboflow

🔗 Dataset Link: https://app.roboflow.com/almas-mnl25/bag-detection-nmfca/1
